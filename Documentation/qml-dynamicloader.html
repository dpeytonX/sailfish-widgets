    Signal Documentation

    create(Component component, variant parent, variant properties)

    Call this signal to begin object creation. Pass in the QML Component, its parent, and properties. onCreate


    error(string errorString)

    Emitted when an error has occurred with object creation. onError


    objectCompleted(variant object)

    Emitted when the object creation has completed. onObjectCompleted
