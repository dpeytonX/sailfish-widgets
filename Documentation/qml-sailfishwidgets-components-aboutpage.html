    Detailed Description

    Back to Sailfish Widgets

    Displays a page that summarizes your application. It includes places for your open source project links, copyright information, application data, and so on. Push this onto the pageStack as you would any other page.

    A special note: I was given permission to incorporate this into Sailfish Widgets by Thomas Amler.

    Property Documentation

    aboutContent : alias

    An alias to the SilicaFlickable page container


    aboutTaskList : alias

    An alias to the SilicaFlickable page container


    application : string

    A string with the application's name


    contributors : variant

    A string array of the contributors to this application.


    copyrightHolder : string

    A string representing the individual or organization holding the application's copyright.


    copyrightYear : string

    A string holding the year(s) that the copyright was instituted.


    description : string

    A string describing the purpose of the application.


    headline : alias

    An alias to the PageHeader Component


    icon : string

    A string representing the location of the application's icon. It can be a resource URI.


    licenses : variant

    A string array of licenses in effect for this application.


    pageTitle : alias

    Use this to set the page's title.


    projectLinks : variant

    A string list of URLs for more information about the application.
