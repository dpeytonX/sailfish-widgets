    Detailed Description

    The DefaultLocale class

    Applications usually have a default translation file which is not suffixed with a country/region specifier.

    This special file is given the locale of DEFAULT_APPLICATION_LOCALE and instead of corresponding to a specific language, it just represents the default translation bundle if no others have been installed.
