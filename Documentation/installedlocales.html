    Detailed Description

    The InstalledLocales class

    Gets the locales available to the application for translation.

    This application will retrieve the harbour-*.qm files and strip the locale suffix then convert them into QML objects that client applications can then access and show to the user.

    This class can also inject a default application locale (user specified) so that end-user apps can use a single model to represent all possible language selections to the user.
