    Detailed Description

    The QmlPropertyWrapper class

    This class was created because connecting to a signal against QQmlProperty that is determined at run time is difficult. The simplest way was to create a wrapper (this class) which provides a signal with the QQmlProperty as a parameter so that the receiver slots can perform different actions depending on the QQmlProperty that emitted the original signal.
