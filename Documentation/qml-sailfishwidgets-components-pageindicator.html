Detailed Description

Back to Sailfish Widgets

This component will show a glass item on the right hand side of a parent (page) along with a label indicating an option for the user to select. Once the user has selected an item either an accepted() or rejected() will be emitted.

Placing multiple instances on a page can provide the user multiple options with which to accept a page without the need for extra buttons, pulley menu items, or toggle switches.

Property Documentation

acceptPoint : real

Sets the width of the swiping area before emmitted the accepted() signal. By default is is two-thirds of the available page width.


enabled : bool

Sets whether this page indicator takes user input.


page : variant

Sets the page in which the indicator should manipulate. It defaults to parent.


text : alias

Alias for the text that appears for the indicator's label.


Signal Documentation

accepted()

Emitted when the user has swiped the indicator past the acceptPoint.


rejected()

Emitted when the user has released the indicator back to neutral position.


Method Documentation

reset()

Resets the accepted status and returns the page back to full opacity.
