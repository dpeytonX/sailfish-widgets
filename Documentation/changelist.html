    SailfishWidgets 3.4

    • Added AboutDialog
    • Added new createObj signal to DynamicLoader
    • Updated SailfishWidgets to Silica 1.1, QtQuick2.2, and Qt 5.6

    SailfishWidgets 1.4.1

    • Added OrientationDialog and OrientationPage will not support inverted portrait by default.

    SailfishWidgets 1.4.0

    • Added OrientationHelper to force Sailfish Silica components' spawned Pages to inherit allowedOrientation
    • Added OrientationDialog and OrientationPage
    • BusyPage FileSelector and AboutPage will support orientation switching by default. To preserve old behavior (no switching) then use version 1.3 of those components.

    SailfishWidgets 1.3.4

    • InstalledLocales
    • LocaleItem
    • DefaultLocale
    • SailfishMain
    • Added isValid() method to ApplicationSettings
    • Added new constructor to ApplicationSettings
    • Added settings() and value() methods to ApplicationSettings
    • Added install and make scripts for automated builds
    • Added debug vs release build command for the make script
    • Release tarball will include the header files
    • SQLiteDatabase has rollback() method
    • Simplified SQLiteDatabase so that only a single Query object is maintained
    • Fixed bug in Qt 5.1 builds from using two internal Query objects
    • Added finish() , close() , lastQuery, executedQuery, fieldSize to Query

    SailfishWidgets 1.3.3

    • BusyPage will allow any orientation
    • LabelButton now has the truncationMode property

    SailfishWidgets 1.3.2

    • Added PageIndicator

    SailfishWidgets 1.3.1

    • Added SQLiteDatabase and Query to the Database package
    • Fixed some warnings with ApplicationSettings, File, and Dir's documentation.
    • Added open() close() readAll() and write() methods to File
    • Added remove() rename() copy() methods to File

    SailfishWidgets 1.3

    • Deprecating Heading in favor of SectionHeader. From now on, use Sailfish.Silica::SectionHeader.
    • Added AboutPage
    • Fixed error in DynamicLoader when accessing errorString on null Component

    SailfishWidgets 1.2

    • Updated StandardListView to remove PageHeader (makes it more useful in dialogs)
    • Added Variant for operations on variant types
    • Added Dir, File, and FileSelector inside the FileManagement package
    • ApplicationSettings will emit settingsPropertyUpdated(QString name) when a property is changed.
    • ApplicationSettings will propagate settings changes to all known QML objects refering to the same settings file.
    • Added Spacer, LabelButton, and BusyPage components

    SailfishWidgets 1.1.2

    • Added ApplicationSettings plugin
    • Added this Change List

    SailfishWidgets 1.1.1

    • Added RadioGroup

    SailfishWidgets 1.1

    • Modularized QML types into directories based on utility
    • Renamed DescriptiveText to Paragraph
    • Started QDoc documentation
