    Detailed Description

    Back to Sailfish Widgets

    This will display a set (of up to 4) TextSwitches in a single row. Designed to allow only one selection to be used at a time.

    Property Documentation

    defaultIndex : TextSwitch

    Set to the TextSwitch which should hold the default check.


    radio1 : alias

    Alias to the TextSwitch at index 1.


    radio2 : alias

    Alias to the TextSwitch at index 2.


    radio3 : alias

    Alias to the TextSwitch at index 3.


    radio4 : alias

    Alias to the TextSwitch at index 4.


    radioMargin : real

    Controls the rightMargin property of all TextSwitches in this RadioGroup.


    read-onlyradiosVisible : int

    Read-only property of the total number of TextSwitches that are visible.


    Signal Documentation

    radioClicked(TextSwitch radio)

    Emitted when a TextSwitch in this group has been clicked. The only parameter is a reference to the TextSwitch component. onRadioClicked


    radioStateChanged(TextSwitch radio)

    Emitted when a TextSwitch in this group has changed checked state. The only parameter is a reference to the TextSwitch component. onRadioStateChanged
