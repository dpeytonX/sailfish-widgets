Detailed Description

Back to Sailfish Widgets

By inserting an OrientationHelper into your Page or Dialog, you can make any other pages spawned from it to inherit the allowedOrientation property.

This is most useful for Sailfish Silica components like ComboBox or TimePickerDialog where you do not have access to the internal Page that the user interacts with.

OrientationHelper {
  excludes: ["_my_page"]
}

Dialog {
  id: t
} // This will inherit the orientation settings of parent

Component {
  id: myPage
  Page {
    property int _my_page
  }
}

Property Documentation

excludes : var

This holds the string properties which if exists serve to exclude child Pages from inheriting the parent's orientation settings.
