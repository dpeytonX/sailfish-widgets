    Detailed Description

    The SailfishMain namespace

    This namespace supports the quick construction of SailfishApps.

    Usage of this library requires that you define the following in the RPM spec file's

    >> macros ... << macros

    section.

      %define __requires_exclude ^libapplicationsettings|libcore.*$

    bool SailfishMain::installLanguage(const QString & appName, const QString & locale, QGuiApplication * app)

    Installs the locale for the given appName of the application app.

    Returns true if successful.

    int SailfishMain::main(int argc, char *[] argv, const QString & appName = QString( "" ), const QString & settingsFile = QString( "" ), const QString & localeSetting = QString( "locale" ))

    Constructs a Sailfish application using appName and settingsFile to load the application settings with a given localeSetting property. If no locale was discovered, the default translation bundle is loaded using appName .qm file.

    Finally, a Sailfish application is constructed using the argc and argv parameters.
